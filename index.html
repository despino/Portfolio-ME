Discover Hawaii

Discover Hawaii through a travel website.

Tic-Tac-Toe

A JavaScript Game.

Want to play Tic-Tac-Toe? This game is built with HTML/CSS, JavaScript, jQuery, and Bootstrap.

RULES OF THE GAME: The computer randomly determines who starts. Players can enter their name. When an empty square is clicked, it is recorded as a play for whoever's turn it is. The side pannels will highlight on the player's turn. The players will alternate turns until a winner reaches 3 'X's or 'O's in a row. The game is responsive and will tell you if a square is already filled; if it's a "Cat's" game; or if there is a winner.

  • Date: October 2015
  • Category: Javascript Game
